TomoTrip-LP の協賛店導線を整理して、同じ入口に何度も戻って見える不自然さを解消してください。

【目的】
- 「協賛店として進む」→ 協賛店向けLPセクションへスクロールしたあと、また同じような入口CTAが連続して見える問題を解消する
- 協賛店向けセクション内の文言を、LP全体の柔らかいトーンに合わせて整理する
- LINE / 公式アプリ / LP の役割が混ざって見えないようにする

--------------------------------------------------
【修正したい問題】
現状では、
1. モーダルで「協賛店として進む」を押す
2. 協賛店セクションへスクロールする
3. その先でまた「登録」「公式アプリで1分登録」など強いCTAが連続して見える
ため、ユーザーが「また同じ入口に戻ってきた」ように感じます。

さらに、協賛店向けセクション内に
- 「LINEで登録」
- 「公式アプリで1分登録」
- 「協賛店として登録する（公式アプリへ）」
など、今のLP方針より強い表現が残っています。

--------------------------------------------------
【修正方針】
1. 協賛店カードのスクロール導線は維持
2. ただし、スクロール先で「次にやること」が自然に1つ見えるよう整理
3. 協賛店向けセクション内のCTA文言をやわらかくする
4. 3ステップ説明の文言も、今のLPトーンに合わせる

--------------------------------------------------
【具体的な文言修正案】

■ 協賛店向け3ステップ
変更前：
- LINEで登録
- プランを作成
- 予約が入る

変更後候補：
- LINEで相談・確認
- 内容を整理する
- 来店や案内につながる

※ 既存の流れや意味を見ながら、自然な表現に整えてください
※ 3ステップ全体の意味が崩れないようにしてください

--------------------------------------------------
■ 協賛店向け説明文内
強めの表現が残っていれば、以下の方向で整えてください。

例：
- 「報酬は自動で計算され、アプリで管理できます」
  → 「活動状況はアプリで確認できます」
  または
  → 「詳しい内容はご案内後に確認できます」

--------------------------------------------------
■ 協賛店向けセクションの大CTA
変更前例：
- 協賛店として登録する（公式アプリへ）

変更後候補：
- まずは内容を確認する
- 協賛店向けの案内を見る
- ご相談・案内へ進む

LP全体のトーンに最も自然なものを選んでください。

--------------------------------------------------
■ 下部固定CTA
変更前例：
- 公式アプリで1分登録

変更後候補：
- まずは内容確認へ進む
- ご利用の流れを確認する
- あなたに合った始め方を選ぶ

--------------------------------------------------
【UI/UX面の改善】
- 協賛店カードクリック後、モーダルを閉じて協賛店セクションへスムーズスクロールする既存動作は維持
- スクロール先に到着したことが分かるよう、必要であれば軽い視覚的補助を追加してもよい
  例：
  - セクション先頭CTAの軽い強調
  - 一時的なハイライト
- ただし派手すぎる演出は不要
- 「また同じ入口が続いている」と見えないよう、文言の役割を整理する

--------------------------------------------------
【重要ルール】
- 既存レイアウトを大きく壊さない
- class / id / data-* / 既存JSはできるだけ維持
- 観光客/ガイド導線には影響を出さない
- 動画導線はそのまま任意の補助導線として残す
- 決済会社審査に不利な強い表現は避ける
